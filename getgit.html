App Features
  • Injects content from any public GitHub repository into WordPress blog posts.
  • Allows for customization of plugin shortcode to prevent conflicts with other installed plugins.
  • Provides caching support through the WordPress Transients API.
  • Caching support is configurable, allowing site operators to enabled caching for shortened page load times, or disable caching to reduce the data store size.
Development Highlights
  • Thoroughly documented with both PHPDoc and inline comments.
  • Integrates with the Sunlight JS syntax highlighter.
  • Ant build and deploy system.
  • Configurable debug logging.
  • Follows WordPress Plugin best practices, such as:
    • Functionality is encapsulated in classes to avoid namespace conflicts.
    • External resources (JS, CSS) are declared through WordPress API hooks.
    • User input is sanitized before being persisted.
