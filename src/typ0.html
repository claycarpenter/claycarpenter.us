Typ0 Programmer-focused typing tutor that makes a game out of practicing the syntax of the user's favorite programming languages. Users upload source files, and then are tested on a randomized set of lines, competing against the clock to score the most points in the least amount of time. Bonus points are awarded for more complex lines and completing the set of test lines before time expires.

App Features
  • Uses new HTML5 file upload functionality to allow users to test themselves against private files without uploading the content to a public site.
  • Real-time highlighting of errors provides immediate feedback on typing accuracy.
  • Test line randomization allows users to repeat the game using the same lines of code from their source file, while also randomizing the order of the test lines to provide fresh challenge.
  • Throwback design mimics the look of early text-only games.
Development Highlights
  • Relies on a custom publish-subscribe notification system that facilitates loose coupling between the widgets and game logic.
  • Uses the jQuery event handling and DOM manipulation frameworks.
  • Ant build and deploy system.
  • Follows JavaScript best practices, such as:
    • Functionality is encapsulated in classes and namespaces.
    • Object-oriented design with inheritance.
    • Singletons provide global access to application state and publish-subscribe event bus.
  • Before implementation was begun, iteration designs were conceptualized through mockups.
